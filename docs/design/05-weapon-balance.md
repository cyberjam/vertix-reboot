# 무기 밸런스 캘리브레이션 (v0.1)

원본 Vertix.io의 **확인된 수치**(데미지·탄창·HP)를 고정점으로 삼고, **추정 항목**(연사·재장전·사거리)을
역할 분화와 TTK(Time-To-Kill) 모델로 일관되게 확정한다. 원본 클라이언트/영상 데이터를 확보하면
추정 항목을 실측치로 교체한다.

> 신뢰도: **[확인됨]** 공개 위키 명시 · **[확정]** 본 캘리브레이션에서 정한 값(추정 기반).

## TTK 모델

`발사 수 = ceil(targetHP / dmg)`, `TTK = (발사수 - 1) × fireRate` (첫 발 0ms 기준).

| 무기 → 대상 | 발사 수 | TTK |
|------|----:|----:|
| Machine Gun → 100 HP | 4 | 270 ms |
| Machine Gun → 50 HP (Hunter) | 2 | 90 ms |
| Sniper → 100/50 HP | 1 | 0 ms (원샷) |
| Machine Pistol → 100 HP | 9 | 480 ms (탄창 2개 필요) |
| Machine Pistol → 50 HP | 5 | 240 ms (정확히 1탄창) |

- **Machine Gun**: 100 HP를 정확히 4발(원본 확인)에 처치. TTK 270ms — 중거리 표준 DPS.
- **Sniper**: 모든 ≤100 HP 클래스를 1발 처치(원본 확인). 단, 연사 1100ms·재장전 2200ms로 **빗나감 페널티**가 크다.
- **Machine Pistol**: 한 탄창(5발=60 dmg)으로는 풀피 100 HP를 못 죽인다 → Hunter의 **백업 무기**. 약화된 적 마무리/근접 자위용.

## 사거리 티어 (Arena01 2000² 기준)

`Sniper(1700) > Machine Gun(750) > Machine Pistol(520)`

- 맵 교전 거리는 엄폐물 사이 ~300–800px. Machine Gun(750)이 대부분 레인 커버.
- Sniper(1700)는 긴 직선 시야를 지배(거의 맵 횡단). 위치 선정이 핵심.
- Machine Pistol(520)은 근접 한정 — Hunter가 주무기 스나이퍼를 우선 쓰게 유도.

## 연사 티어

`Machine Pistol(60ms) < Machine Gun(90ms) < Sniper(1100ms)`

## 불변식 (회귀 검증 대상)

다음은 캘리브레이션의 핵심 계약이며 변경 시 의도적이어야 한다.

1. `ceil(100 / machinegun.dmg) === 4` (원본 확인 사실)
2. `sniper.dmg >= 100` (100 HP 원샷)
3. `machine_pistol.mag * machine_pistol.dmg < 100` (한 탄창으로 풀피 처치 불가)
4. 사거리: `sniper > machinegun > machine_pistol`
5. 연사: `machine_pistol < machinegun < sniper`

> 본 빌드에서 위 5개 불변식을 단위 검증으로 확인함(통합 스모크 별도).
